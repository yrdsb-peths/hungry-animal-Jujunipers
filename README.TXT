PROJECT TITLE: Hungry Elephant
PURPOSE OF PROJECT: Game
VERSION or DATE: Dec 11 2023
HOW TO START THIS PROJECT:
AUTHORS: Julia, edits by Yuvia, Justin and Tiffany
USER INSTRUCTIONS: Move elephant using left and right keys to collect apples and earn points.
FEATURES:
- Elephant plays walking animation when moving left/right
- Elephant increases in size as more apples are eaten
- Elephant moves faster as more apples are eaten, to compensate for the increase in fall speed of the apples
- Score system to track number of apples eaten
- Lives counter (added by Yuvia)
- Sound effects for lives decrease (added by Yuvia)
- Sound effect 
- Elephant appears on other side of screen when it walks off screen (added by Justin)
- Title screen background (added by Tiffany)
